Python Backend API Shape

This single-page prototype uses ApiClient below. Right now it is backed by mock data, but the method names and payloads are structured so you can swap to fetch(), WebSocket events, or SSE from FastAPI later.

GET    /api/v1/status
GET    /api/v1/timer/state
POST   /api/v1/timer/arm
POST   /api/v1/timer/start
POST   /api/v1/timer/stop
POST   /api/v1/timer/reset
GET    /api/v1/runs/recent?limit=10
GET    /api/v1/leaderboards/today?course_id=speed-gauntlet
GET    /api/v1/leaderboards/all-time?course_id=speed-gauntlet
GET    /api/v1/queue
POST   /api/v1/queue
DELETE /api/v1/runs/{run_id}
WS     /api/v1/ws/live
